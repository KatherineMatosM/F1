# F1 Racing 

Página web interactiva dedicada a la **Fórmula 1** con información sobre escuderías, estadísticas y clasificación de pilotos.

## Descripción

Sitio web moderno y responsive que presenta información sobre el mundo de la F1, incluyendo:

- Información de las principales escuderías  
- Estadísticas de la temporada actual  
- Próximas carreras y circuitos  
- Clasificación de pilotos  
- Formulario de contacto  


## Tecnologías Utilizadas

- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- Font Awesome Icons  

## Características

- Diseño responsive adaptable a todos los dispositivos  
- Animaciones y efectos interactivos  
- Contadores animados en estadísticas  
- Navegación suave entre secciones  
- Paleta de colores inspirada en la F1  

## Instalación Local

Clona este repositorio:

```bash
git clone https://github.com/KatherineMatosM/F1.git
```
Abre el archivo `index.html` en tu navegador.

## Despliegue

Este proyecto está configurado para desplegarse automáticamente en **GitHub Pages**.  
Cualquier cambio en la rama `main` se reflejará automáticamente en el sitio web.
